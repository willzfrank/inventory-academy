This is a React 18 + Vite + TypeScript project.

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing by modifying files under `src/`.

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run start
```
